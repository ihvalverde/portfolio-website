## 📝 Descripción del Proyecto

¡Bienvenido a mi portafolio web! Este espacio ha sido diseñado y maquetado con un enfoque minimalista y elegante para presentar mis habilidades técnicas, mi trayectoria académica, certificaciones profesionales y los sistemas interactivos que he construido.

Cuenta con una interfaz moderna basada en una paleta de colores pastel, alta legibilidad en pantallas móviles (Responsive Design), una estructura puramente semántica y un formulario de contacto directo integrado.

👉 **[Ver el Portafolio en Vivo Aquí](https://portfolio-website-ihvalverdeb.vercel.app/)**

---

## 🚀 Secciones del Portafolio

*   **Sobre mí:** Breve introducción sobre mi enfoque en ingeniería de software y desarrollo de interfaces.
*   **Mis Proyectos:** Módulo interactivo que destaca desarrollos de sistemas de inventario Full Stack, maquetación web y prototipado UI/UX.
*   **Certificaciones:** Galería estructurada con accesos directos a credenciales verificables de tecnología, bases de datos e idiomas.
*   **Habilidades:** Bloque visual con las tecnologías Core que domino.
*   **Formulario de Contacto:** Integración fluida para mensajería directa en tiempo real.

## 📂 Estructura del Repositorio

El proyecto mantiene un orden limpio y modular para facilitar la carga de recursos visuales e imágenes de interfaz:

```text
