## 📝 Descripción del Proyecto

¡Bienvenido a mi portafolio web! Este espacio ha sido diseñado y maquetado con un enfoque minimalista y elegante para presentar mis habilidades técnicas, mi trayectoria académica, certificaciones profesionales y los sistemas interactivos que he construido.

Cuenta con una interfaz moderna basada en una paleta de colores pastel, alta legibilidad en pantallas móviles (Responsive Design), una estructura puramente semántica y un formulario de contacto directo integrado.

👉 **[Ver el Portafolio en Vivo Aquí](https://portfolio-website-ihvalverdeb.vercel.app/)**

---

## 🚀 Secciones del Portafolio

*   **Sobre mí:** Breve introducción sobre mi enfoque en ingeniería de software y desarrollo de interfaces.
*   **Mis Proyectos:** Módulo interactivo que destaca desarrollos de sistemas de inventario Full Stack, maquetación web y prototipado UI/UX.
*   **Certificaciones:** Galería estructurada con accesos directos a credenciales verificables de tecnología, bases de datos e idiomas.
*   **Habilidades:** Bloque visual con las tecnologías Core que domino.
*   **Formulario de Contacto:** Integración fluida para mensajería directa en tiempo real.

## 📂 Estructura del Repositorio

El proyecto mantiene un orden limpio y modular para facilitar la carga de recursos visuales e imágenes de interfaz:

```text
portfolio-website/
├── assets/
│   ├── css/
│   │   └── style.css       # Estilos globales y paleta de colores
│   └── img/                # Fotografías del perfil y capturas de proyectos (.jpeg, .png, .jpg)
├── index.html              # Estructura principal y maquetación general
└── README.md               # Documentación del portafolio

## 🎨 Paleta de Colores y Estilos

El diseño visual está centralizado en la raíz del documento (`:root`) mediante propiedades nativas de CSS para mantener la consistencia estética en toda la aplicación:

*   **Rosa Pastel Principal (`--primary-pink: #f6c4df`):** Utilizado en componentes clave, bordes, cajas de habilidades y botones secundarios.
*   **Rosa de Enfoque (`--secondary-pink: #e8a7c9`):** Aplicado a estados *hover*, títulos interactivos, iconos de certificación y enlaces web.
*   **Fondo Suave (`--light-bg: #fdfbfd`):** Genera un contraste limpio, suave y descansado para la lectura prolongada.
*   **Tipografía:** Integración fluida de **Poppins** (pesos 300, 400, 600) importada directamente de Google Fonts.

## ⚙️ Visualización e Instalación Local

Si deseas clonar este repositorio para examinar el código base de manera local en tu entorno de desarrollo:

1. **Clona el proyecto:**
   ```bash
   git clone https://github.com/ihvalverde/portfolio-website.git
   ```
2. **Navega al directorio:**
   ```bash
   cd portfolio-website
   ```
3. **Ejecuta la interfaz:**
   Abre el archivo `index.html` directamente en tu navegador predeterminado o utiliza la extensión *Live Server* en Visual Studio Code para actualizaciones en tiempo real.
